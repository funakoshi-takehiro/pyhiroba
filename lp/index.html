はじめる

初回のみPython環境の読み込みが必要です（30秒〜1分程度）。
2回目以降はすぐに起動します。

FEATURES

日本の教育現場のための
Python実行環境。

FEATURE / 01

無料で利用可能

広告ゼロ、無料、人数制限も利用制限もありません。

FEATURE / 02

生徒を守る安心設計

生徒の登録やログイン不要、ローカル駆動。生徒が書いたコード・個人情報を一切外部に送信しない設計です。

FEATURE / 03

すぐに授業で使える簡単環境

環境設定不要。ブラウザでURLを開くだけで即使え、校内WiFiに負荷をかけずにクラス全員の一斉実行にも対応します。

FEATURE / 04

生徒の学びをサポートする工夫

URLひとつで教材配布が可能。エラーメッセージを日本語で表示する機能、授業スライドやテキストもあわせて使用できます。

FEATURE / 05

先生の負担を下げる工夫

生徒のアカウント管理不要。URLを共有するだけで授業前の準備が完了し、環境設定や更新作業も必要ありません。

FAQ

よくある質問

Q.本当に無料ですか？広告はありますか？
はい、完全に無料です。広告も、ユーザー登録もありません。学校予算ゼロで、すべての生徒に提供できます。
Q.コードや個人情報はどこに保存されますか？
コードは生徒のブラウザ内（ローカル）でのみ実行・保存されます。外部サーバーに送信される情報は一切ありません。
Q.学校のWi-Fiでクラス全員が同時に使えますか？
はい。初回読み込み後は完全ローカル動作のため、ネットワーク負荷がかかりません。一斉実行にも耐えられます。
Q.使えるライブラリは？
numpy・pandas・matplotlib・scikit-learn・Pillow など、教育で頻出のライブラリを標準搭載。import 文を書けば自動で読み込まれます。
Q.教材はどう配布できますか？
ノートをURLにエンコードして共有できます。生徒はリンクを開くだけ、教材＋コード＋スライドが一体となって届きます。
Q.古い端末や学校のChromebookでも動きますか？
近年のブラウザ（Chrome/Edge/Safari）が動く端末であれば動作します。インストール不要、管理者権限も必要ありません。
