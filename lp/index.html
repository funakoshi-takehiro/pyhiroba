FEATURES

日本の教育現場のための
Python実行環境。

FEATURE / 01

無料で利用可能

完全無料、広告なしでご利用いただけます。人数や利用回数の制限もありません。なお、先生方にはご利用開始前に、簡単なアンケートのご回答をお願いしています。

FEATURE / 02

生徒を守る安心設計

生徒はアカウント登録不要でご利用いただけます。生徒のパソコン上で動作するため、生徒の学習内容は一切外部に送信されません。

FEATURE / 03

授業で使いやすい仕組み

プログラミングのコードだけでなく、授業スライドや説明テキストも同じノートブック上で併用できます。さらに、各端末のブラウザ内で動作するため、校内WiFiに負荷をかけずにクラス全員での一斉実行も可能です。

FEATURE / 04

生徒の学びをサポートする工夫

既存のプログラミング環境では、エラーが長い英語・赤文字で表示され、生徒が苦手意識を持ちやすいという課題があります。PyHirobaは、エラー内容をやさしい日本語のヒントで表示し、つまずきを学びに変えます。対応エラーの一覧はこちら →

FEATURE / 05

先生の負担を下げる工夫

生徒のアカウント管理は不要。URLを共有するだけで授業前の準備が完了し、環境設定や更新作業もいりません。さらに、公開されている既存教材はもちろん、自作した教材も使用可能です。

公開教材の一覧 → 自作教材の公開方法 →
FEATURE / 06

だれもが使えるアクセシビリティ

キーボードだけでの操作、スクリーンリーダーによる読み上げ、見やすい色のコントラスト、目にやさしいダークモードに対応。さまざまな環境・特性の生徒が、安心して学べる学習環境を目指しています。
