FEATURES

日本の教育現場のための
Python実行環境。

FEATURE / 01

無料で利用可能

完全無料、広告なしでご利用いただけます。人数や利用回数の制限もありません。

FEATURE / 02

生徒を守る安心設計

生徒はアカウント登録不要でご利用いただけます。生徒のパソコン上で動作するため、生徒の学習内容は一切外部に送信されません。

FEATURE / 03

すぐに授業で使える簡単環境

ブラウザでURLを開けばすぐにご利用いただけます。また、校内WiFiに負荷をかけずにクラス全員の一斉実行も可能です。

FEATURE / 04

生徒の学びをサポートする工夫

URLひとつで教材配布が可能。エラーメッセージを日本語で表示する機能、授業スライドやテキストもあわせて使用できます。

FEATURE / 05

先生の負担を下げる工夫

生徒のアカウント管理不要。URLを共有するだけで授業前の準備が完了し、環境設定や更新作業も必要ありません。
