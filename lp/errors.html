ERROR GUIDE

日本語表示対応エラーの一覧

PyHiroba（ぱいひろば）は、Pythonの実行中に発生したエラーを自動で読み取り、 初学者にもわかりやすい日本語のヒントで表示します。

25 エラー種類
80+ 対応パターン

文法・字下げのエラー

SyntaxError / IndentationError / TabError
エラー種別よくある原因日本語での解説（要約）
SyntaxErrorif x > 0 のようにコロンを忘れたコロン「:」が必要です。if・for・def などの行末には必ず「:」を付けてください。
SyntaxError全角の記号やスペースが混入した使えない文字が含まれています。括弧・コロン・イコールなどの記号は半角で入力してください。
SyntaxErrorクォートを閉じ忘れた文字列が閉じられていません。「'」や「"」の対応を確認してください。
SyntaxError括弧 () [] {} の対応ミス括弧が閉じられていません。開いた括弧と閉じた括弧の数を確認してください。
SyntaxErrorf"値は{}" のように {} が空f文字列の { } の中が空です。変数名や式を書いてください。
SyntaxErrorif x = 5: と書いた代入できない場所に「=」があります。比較は「==」を使ってください。
SyntaxErrorreturn/break を関数やループの外で使ったreturn は関数の中、break・continue は繰り返しの中でのみ使えます。
SyntaxErrorprint "x" と古い書き方をしたPython3 では print は ( ) で囲みます。print("x") としてください。
SyntaxError[1 2 3] のようにカンマを忘れたカンマ「,」を書き忘れています。要素どうしは , で区切ります。
SyntaxError括弧の種類 () [] {} が食い違う開いた括弧と閉じた括弧の種類が合っていません。種類をそろえてください。
IndentationError字下げ幅が前の行とそろっていない字下げの位置が合っていません。同じブロックは同じスペース数（4つ）にそろえてください。
IndentationError字下げが必要な行を字下げしていないインデントが必要です。if・for・def の次の行はスペース4つで字下げしてください。
IndentationError余分なスペースが入っている余分なインデントがあります。行の先頭の空白を確認してください。
TabErrorタブとスペースが混在しているタブとスペースが混在しています。インデントはスペース4つで統一してください。

名前・変数のエラー

NameError / UnboundLocalError
エラー種別よくある原因日本語での解説（要約）
NameError変数・関数のスペルミスその名前が定義されていません。スペルミスや、定義したセルを先に実行したか確認してください。
NameErrorPrint() のように大文字で書いたprint は全部小文字です、など、よくある書き間違いも個別にヒント表示します。
NameErrortrue / null と書いたPythonでは True・False・None（先頭大文字）を使います。
UnboundLocalError関数内で定義前の変数を使った関数の中で、値を代入する前にその変数を使っています。先に代入するか global 宣言してください。

データ型のエラー

TypeError
エラー種別よくある原因日本語での解説（要約）
TypeError5 + "3" のように数値と文字列を計算型が違うため計算できません。int()・str() で変換してください。
TypeError"点数" + 85 と結合した文字列と数値は「+」でつなげません。str(85) や f文字列を使ってください。
TypeError関数に渡す引数の数が違う引数の数が合いません。関数の定義と呼び出し方を確認してください。
TypeErrorfor i in 5: と書いたその型は繰り返せません。range(5) などを使ってください。
TypeError変数名が組み込み関数と衝突関数ではないものを () で呼び出しています。list などの変数名を避けましょう。
TypeErrorNone を繰り返し処理に使った関数が値を return しているか確認してください。
TypeError"5" > 3 と文字列と数値を比較型が違うものは大小を比べられません。int() でそろえてください。
TypeErrorrange(3.0) と小数を渡した整数が必要な場所に小数を渡しています。range(3) としてください。
TypeError"-".join([1,2,3]) で数値を結合join でつなげられるのは文字列だけです。str() で変換してください。
TypeErrors["a"] と文字列に文字で添字文字列・リストの [ ] には整数の番号を使います。

値のエラー

ValueError
エラー種別よくある原因日本語での解説（要約）
ValueErrorint("abc") と変換した文字列を整数に変換できません。数字だけの文字列か確認してください。
ValueErrora, b = [1, 2, 3] と展開した左辺の変数の数と右辺の要素の数を合わせてください。
ValueErrormath.sqrt(-1) を計算した数学的に定義できない計算です（負の数の平方根など）。入力値を確認してください。
ValueErrorグラフのx・yのデータ数が違うx軸とy軸のデータ数が一致しません。同じ長さのデータを使ってください。
ValueErrorNumPy配列の形が合わない配列の形（shape）が合いません。.shape で確認してください。
ValueErrormax([]) と空リストを渡した空のリストには max()・min() は使えません。中身があるか len() で確認してください。

リスト・辞書の参照エラー

IndexError / KeyError
エラー種別よくある原因日本語での解説（要約）
IndexErrorリストの範囲外の番号を指定範囲外にアクセスしています。最初は [0]、インデックスは 0〜len()-1 の範囲です。
IndexErrorNumPy配列の範囲外アクセス配列のサイズを超えています。.shape で形を確認してください。
KeyError辞書に存在しないキーを指定そのキーは辞書にありません。スペルを確認するか .get() を使ってください。

計算・再帰のエラー

ZeroDivisionError / RecursionError / OverflowError
エラー種別よくある原因日本語での解説（要約）
ZeroDivisionError0で割り算した0（ゼロ）で割っています。分母が0でないか確認してください。
RecursionError再帰関数に終了条件がない再帰が無限ループになっています。必ず終了条件を書いてください。
OverflowError数値が大きくなりすぎた数値が大きすぎて処理できません。計算の途中を確認してください。

ライブラリ・属性のエラー

ModuleNotFoundError / ImportError / AttributeError
エラー種別よくある原因日本語での解説（要約）
ModuleNotFoundErrorライブラリ名のスペルミスそのライブラリが見つかりません。使える主なライブラリ名を一緒に表示します。
ImportErrorモジュール内の名前を間違えたそのモジュールに指定した名前はありません。スペルを確認してください。
AttributeErrorメソッド・属性のスペルミスその属性・メソッドはありません。スペルや、データの型を確認してください。
AttributeErrorx = lst.sort() の戻り値を使ったNone に属性を使おうとしています。メソッドの戻り値が None でないか確認してください。

ファイル・その他のエラー

FileNotFoundError / AssertionError / RuntimeError ほか
エラー種別よくある原因日本語での解説（要約）
FileNotFoundErrorファイルを読み込もうとしたファイルが見つかりません。ブラウザ上のPythonではローカルファイルは扱えません。
AssertionErrorassert の条件を満たさない検証（assert）が失敗しました。条件式を確認してください。
RuntimeErrorループ中に辞書を変更した実行時エラーです。ループ中のデータ変更などを見直してください。
MemoryError巨大なデータを作ろうとしたメモリが不足しています。より小さなデータで試してください。
StopIterationnext() の使いすぎイテレータの要素がなくなりました。for文を使うとより安全です。
上記以外のエラーも、英語の原文・エラー種別・発生した行番号とともに表示します。
該当しないエラーには「エラーメッセージをよく読んでみよう」という汎用のヒントを表示します。
