プライバシーポリシー

PyHiroba（ぱいひろば、以下「本サービス」）における、利用者の情報の取り扱いについて定めます。本サービスは、利用者のプライバシーを最大限に尊重し、原則として個人情報を収集しません。

最終更新日：2026年7月1日

1. 収集しない情報

本サービスは、次の情報を 一切収集・保存しません。

  • 氏名・メールアドレス・学校名などの個人情報
  • アカウント情報（本サービスにアカウント登録はありません）
  • 入力・作成したコードやノートブックの内容（後述のとおり、すべて利用者の端末内で処理されます）

2. Cookie・アクセス解析について

本サービスは、Cookie による利用者の追跡や、アクセス解析ツール（Google Analytics 等）による行動の記録を 行いません。

3. コードの実行について

本サービス上で書いた Python のコードは、利用者自身のブラウザ内（お使いの端末の中）で実行されます。コードや実行結果が本サービスのサーバーへ送信・保存されることはありません。

4. 外部サービスの利用について

本サービスは、表示や機能の提供のために、次の外部サービスを利用します。これらの外部サービスにアクセスする際、技術的な情報（IP アドレス・ブラウザの種類など）が各サービスの提供者に送信されることがあります。

  • コンテンツ配信（CDN）：Python 実行環境や表示用ライブラリ・フォントの読み込み（jsDelivr、Cloudflare、Google Fonts 等）
  • Google Drive / GitHub 等：URL やファイル ID から外部の教材（.ipynb）を読み込む機能を利用した場合
  • お問い合わせフォーム：Google フォームを利用しています。フォームから送信された情報は Google のプライバシーポリシーに従って取り扱われます

これらの外部サービスにおける情報の取り扱いについては、各サービス提供者のプライバシーポリシーをご確認ください。

5. 未成年の方の利用

本サービスは学校教育での利用を想定しています。本サービスは個人情報を収集しませんが、外部の教材やリンクには第三者が作成した内容が含まれる場合があります。心当たりのないリンクは開かないなど、安全にご利用ください。

6. 本ポリシーの変更

本ポリシーは、必要に応じて変更することがあります。変更後の内容は、本ページに掲載した時点から効力を生じます。

7. お問い合わせ

本ポリシーに関するご質問は、お問い合わせフォームまでお願いいたします。あわせて利用規約もご確認ください。
