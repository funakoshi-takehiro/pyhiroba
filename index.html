FEATURES

日本の教育現場のための
Python実行環境。

FEATURE / 01

無料で利用可能

広告ゼロ、完全無料。人数・利用制限もありません。学校予算ゼロで、すべての生徒に提供できます。

FEATURE / 02

生徒を守る安心設計

登録・ログイン不要、ローカル駆動。生徒が書いたコードも個人情報も、一切外部サーバーに送信されません。

FEATURE / 03

すぐに授業で使える簡単環境

環境設定不要。ブラウザでURLを開くだけで即使え、校内WiFiに負荷をかけずにクラス全員の一斉実行にも対応します。

FEATURE / 04

生徒の学びをサポートする工夫

URLひとつで教材配布が可能。エラーを日本語でわかりやすく表示し、授業スライドやテキストもコードと一体で使用できます。

FEATURE / 05

先生の負担を下げる工夫

アカウント管理不要。URLを共有するだけで授業前の準備が完了。ソフトウェアの更新作業も一切不要です。

FAQ

よくある質問

Q.本当に無料ですか？広告はありますか？
はい、完全に無料です。広告も、ユーザー登録もありません。学校予算ゼロで、すべての生徒に提供できます。
Q.コードや個人情報はどこに保存されますか？
コードは生徒のブラウザ内（ローカル）でのみ実行・保存されます。外部サーバーへ送信される情報は一切ありません。
Q.学校のWi-Fiでクラス全員が同時に使えますか？
はい。初回読み込み後は完全ローカル動作のため、ネットワーク負荷がかかりません。一斉実行にも対応しています。
Q.使えるライブラリは何ですか？
numpy・pandas・matplotlib・scikit-learn・Pillow など、教育で頻出のライブラリを標準搭載。import 文を書けば自動で読み込まれます。
Q.古い端末や学校のChromebookでも動きますか？
近年のブラウザ（Chrome / Edge / Safari）が動く端末であれば動作します。インストール不要、管理者権限も必要ありません。
